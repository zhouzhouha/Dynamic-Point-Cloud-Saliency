Visual Saliency of Dynamic Point Cloud

A companion project page for two studies on visual attention in 6‑DoF VR: task‑dependent saliency and a complementary task‑free dataset & analysis. Click the teaser image to switch between conditions.

Read the papers See the teaser

What is this about?

This page presents two tightly related efforts studying visual saliency for dynamic point clouds in 6‑DoF virtual reality. The first work focuses on task‑dependent viewing, where participants perform explicit tasks while we capture gaze and head motion. The second introduces a task‑free dataset and comparison—revealing how attention patterns differ when observers freely explore content without a task. Together, these studies provide actionable insights for PCQA, compression, streaming, and interaction design.
